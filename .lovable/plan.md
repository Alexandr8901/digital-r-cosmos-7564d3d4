### Блок C. Кабинет гражданина
- Layout `/_authenticated/citizen/route.tsx` с Sidebar (shadcn sidebar) — Главная, Кошелёк, Платежи, Переводы, QR, История, Аналитика, Подписки, Документы, Уведомления, AI, Профиль, Настройки, Поддержка. Topbar: поиск, тема, уведомления, профиль-меню.
- Все страницы — на mock-данных в `src/lib/mock/citizen.ts` (детерминированный сид). Реальные транзакции не выдумываем как «уже работающие банковские подключения» — статус «демо-данные» в углу.

Страницы:
1. **Dashboard** `/citizen` — баланс, доходы/расходы (sparkline), последние операции, избранные получатели, автоплатежи, подписки, уведомления, быстрые действия (10 кнопок).
2. **Кошелёк** `/citizen/wallet` — общий баланс, карточки счетов, цифровой рубль (статус «после подключения официального API»), лимиты, безопасность.
3. **Платежи** `/citizen/payments` — ЖКХ, интернет, мобильная связь, штрафы, налоги; форма + история.
4. **Переводы** `/citizen/transfers` — табы: между своими / по телефону / по реквизитам / по QR / по шаблону; автопереводы.
5. **QR** `/citizen/qr` — показать свой QR (генерация через `qrcode.react`), сканер-плейсхолдер, история, избранные.
6. **История** `/citizen/history` — таблица с поиском, фильтрами (период / категория / тип / статус), экспортом CSV.
7. **Аналитика** `/citizen/analytics` — recharts: pie (категории), line (месяцы), bar (топ-платежи), KPI-карточки, прогноз.
8. **Подписки** `/citizen/subscriptions` — список с датой/стоимостью/автопродлением, действия.
9. **Документы** `/citizen/documents` — выписки/квитанции/чеки, поиск, экспорт PDF (заглушка).
10. **Уведомления** `/citizen/notifications` — лента событий, фильтр, отметка прочитанного.
11. **AI** `/citizen/ai` — чат на AI Elements + Lovable AI Gateway (`google/gemini-2.5-flash`), 8 быстрых команд, история в localStorage.
12. **Профиль** `/citizen/profile` — данные из `profiles`, форма редактирования, безопасность, сессии (заглушка), уведомления.
13. **Настройки** `/citizen/settings`, **Поддержка** `/citizen/support`.

Везде: skeleton-загрузка, Empty State, Error State (TanStack errorComponent), Success toasts (sonner), темы, полная адаптивность, мобильный bottom-nav.

## Технические детали

- **Стек**: TanStack Start + React 19 + Tailwind v4 + shadcn/ui + framer-motion + recharts + AI Elements + Lovable Cloud (Supabase).
- **Routing**: всё под `src/routes/`. Защищённые экраны под `_authenticated/citizen.*`. Публичные страницы — top-level. Каждый маршрут — свой `head()` (title/description/og).
- **State**: TanStack Query для server fns, локальные mock-данные через детерминированные генераторы.
- **Server fns**:
  - `getProfile`, `updateProfile`, `setActiveRole` (`requireSupabaseAuth`).
  - `aiChat` — стрим через AI Gateway (server route `src/routes/api/citizen-ai.ts`).
- **DB-миграция** (одна):
  - Расширить `profiles`: `middle_name`, `address`, `notifications_settings jsonb`, `passkey_enabled bool`.
  - Таблица `user_settings` (KV jsonb per user) с RLS + GRANT.
  - Таблица `audit_log` (журнал действий) с RLS (`SELECT own only`, `INSERT через service_role`).
- **Sidebar**: `@/components/ui/sidebar` (shadcn), активный пункт по `useRouterState`.
- **Mock**: модули `mock/transactions.ts`, `mock/subscriptions.ts`, `mock/notifications.ts` — детерминированный seedrandom.
- **AI Gateway**: модель `google/gemini-2.5-flash`, system prompt — финансовый ассистент ЦифроРубля; примеры/быстрые команды на русском.
- **Honesty**: интеграции и банковские подключения визуально помечены статусами «доступно / в разработке / после официального API». Никаких выдуманных «уже работает».
- **A11y**: focus rings, aria-labels, keyboard nav в sidebar и мастере регистрации.
